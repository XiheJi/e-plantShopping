# e-plantShopping

**Repository name:** `e-plantShopping`
**Project name:** Paradise Nursery — Plant Shopping App

## What is this project?

This is the **e-plantShopping** project: a simple online shopping website for a
houseplant store called **Paradise Nursery**. It lets you browse plants and buy
them, all from your web browser.

In plain words, the app works like a small online plant shop:

- You land on a welcome page that tells you about the store.
- You click a button to start shopping.
- You look through different kinds of houseplants and add the ones you like to a
  shopping cart.
- You open the cart to see what you picked, change how many of each plant you
  want, remove plants you no longer want, and see the total price.

## The three pages

1. **Landing page** — A welcome screen with a background image, the company name
   (Paradise Nursery), a short paragraph about the company, and a **Get Started**
   button that takes you to the plants.
2. **Product listing page** — Shows the houseplants for sale, grouped into
   categories. Each plant has a picture, a name, a price, and an **Add to Cart**
   button. A header at the top shows a cart icon with the number of items in your
   cart and lets you move between pages.
3. **Shopping cart page** — Shows every plant you added, with its picture, name,
   and price. You can increase or decrease how many of each plant you want,
   delete a plant, and see the total number of items and the total cost. It also
   has a **Continue Shopping** button and a **Checkout** button.

## Built with

- **React** — for building the user interface.
- **Redux Toolkit** — for keeping track of what is in the shopping cart.
- **Vite** — for running and building the app.

## Running the app locally

```bash
npm install   # install the dependencies
npm run dev   # start the app for development
npm run build # create a production build
```
